# ⚡ India Flash News

![Banner](frontend/assets/banner.png)

A professional, high-performance news application delivering real-time updates from across India. Featuring multi-language support (English, Hindi, Marathi) and district-level precision.

## ✨ Features

- 🇮🇳 **Hyper-Local Focus**: Search news for any specific Indian district.
- 🌍 **Multi-Language**: Instant switching between English, Hindi, and Marathi.
- ⚡ **Real-Time Updates**: Auto-refresh functionality for the latest stories.
- 🔖 **Smart Bookmarks**: Save important articles directly to your local storage.
- 🔍 **Hybrid Search**: intelligent fallback between DuckDuckGo News and NewsAPI for maximum coverage.
- 🎨 **Premium UI**: Modern dark-mode interface with smooth animations and responsive design.

## 🛠️ Local Installation

### Setup
```bash
# Install dependencies
pip install -r backend/requirements.txt

# Run locally
python backend/index.py
```
Open `http://localhost:5000` in your browser.
